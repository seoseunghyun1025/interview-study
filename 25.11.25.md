# SSAFY 면접 준비 정리 (2025-11-25)

## 1. SSAFY가 보는 포인트 정리

### 1) 인재상 / 평가 항목
- **SSAFY 인재상**
  - 논리적 사고
  - 열정
  - 학습 의지

- **SSAFY 인터뷰 평가 항목**
  - **IT관심도**: 일상에서 IT/디지털 이슈를 얼마나 이해하고 관심 갖고 있는지
  - **사고 / 추론**: 사회 이슈를 논리적으로 정리하고, 문제 → 원인 → 해결 방안으로 풀어내는 능력
  - **열정(학습 의지)**: 1년 동안 교육을 끝까지 수행할 수 있는 책임감과 꾸준함

→ 결론: **깊은 전문 지식보다, 태도/성실/논리 + IT 관심도를 더 중요하게 본다.**

---

## 2. 인성 면접 방향성

슬라이드 요약:

- 직무 전문성보다는 **성실성, 태도, 팀워크**를 평가하는 경향이 강함  
- 인성 질문 약 **15~16개** 정도로 구성  
- 어려운 꼬리 질문보다는  
  - 일반적인 자기소개  
  - 협업 경험  
  - 성장 이유 / 지원 동기  
  등 “공식 질문” 중심

→ **내 경험을 자연스럽게 풀어내는 연습**이 가장 중요함.

오늘 정리한 내 경험 키워드:
- 졸업프로젝트: 자취 커뮤니티 서비스 **StackFlov 백엔드 개발**
- WebSocket 채팅 구현 실패 경험 → “구조 이해의 중요성” 깨달음
- 군 제대 후 **매일 10분 달리기**로 꾸준함/셀프관리
- 학교 **도서관 IT 근로** 경험 (반복 업무, 책임감, 성실함)
- SSAFY 면접 스터디 직접 개설 → 주도성, 학습 의지

## 3. PT 면접(IT 이슈) 대비 정리

### 1) PT 면접 형식 이해
- 당일에 **사회/IT 이슈 기사 + 문제 + 키워드**가 적힌 문제지를 받음
- 정해진 시간 동안 준비 후, **3~5분 발표**
- 제시된 **키워드 2개 이상**을 포함해서  
  - 기사 요약  
  - 문제점  
  - 해결 방안  
  - 내 생각/경험  
  을 논리적으로 말해야 함

### 2) 만능 답변 구조(오늘 정리한 템플릿)

1. **이슈/기사 요약**  
   - “어떤 내용인지 한 줄로 설명”

2. **문제점 or 쟁점 2개**
   - 사회적/기술적 문제를 2가지로 나눠서 정리

3. **해결 방안 + 제시 키워드 포함**
   - 교육, 정부 지원, 기업 책임, 기술 활용 등 키워드를 자연스럽게 문장에 넣기

4. **내 경험 + 개발자로서 태도**
   - StackFlov, GPT 활용 경험, 달리기/근로 경험 등과 연결
   - “그래서 SSAFY에서 ~를 이렇게 배우고 싶다”로 마무리

## 4. 공부 우선순위 정리

- **PT & 인성** : **7**
  - IT 기사 읽기 + 위 구조로 발표 연습
  - 자기소개, 지원 동기, 협업/실패/성장 경험 스토리 정리

- **CS 기초** : **3**
  - 깊게 말고, 아래 정도만 “한 줄로 설명” 가능하게 준비
    - 프로세스 vs 스레드, 스택/큐
    - HTTP vs HTTPS, REST API
    - 배열/리스트/해시맵, 시간 복잡도 개념
    - PK/FK, 인덱스, 트랜잭션

## 5. 면접 스터디 운영 아이디어 (메모)

- 주 2회 기준
  - 1시간: PT 연습 (기사 선택 → 15~20분 준비 → 3분 발표 + 피드백)
  - 1시간: 인성 질문 롤플레잉 + CS 기초 질답
- 스터디에서 공통으로 다룰 질문
  - 1분 자기소개 / SSAFY 지원 동기
  - 협업 경험, 갈등 해결 경험
  - 힘들었던 경험과 극복 과정
  - 꾸준히 해온 것 / 1년 과정 버틸 수 있는 이유
